1. 개요

PwNote는 오프라인 비밀번호 관리자입니다. 민감한 데이터는 기기에 로컬 저장되며, 계정/클라우드 동기화 없이 마스터 비밀번호로 암호화 보호됩니다.

중요: PwNote는 보관함, 마스터 비밀번호, 복구 키에 접근할 수 없습니다. 둘 다 분실하면 데이터는 복구할 수 없습니다.

2. 시작하기

2.1 첫 실행

  • 처음 실행 시 마스터 비밀번호를 설정하세요.
  • PwNote는 복구 키를 생성하고 한 번만 표시합니다.
  • 복구 키를 안전한 곳에 보관하세요.
  • 설정 후 카테고리와 항목을 생성할 수 있습니다.

2.2 무료 / Pro

무료 버전 제한:

  • 카테고리 5개까지
  • 항목 30개까지

Pro(인앱 구매) 기능:

  • 무제한 항목
  • Face ID 빠른 잠금 해제
  • 비밀번호 생성기
  • 복구 키 재생성 (기존 키 무효화)

3. 백업 및 복원

PwNote는 암호화된 백업 파일의 내보내기/가져오기를 지원합니다. iOS 파일 앱에서 저장 위치를 직접 선택합니다.

3.1 암호화 백업 만들기

  1. PwNote를 열고 마스터 비밀번호로 잠금 해제합니다.
  2. 설정의 백업 / 내보내기로 이동합니다.
  3. 암호화 백업 내보내기를 선택합니다.
  4. 파일 시트에서 폴더를 선택해 저장합니다.

3.2 암호화 백업에서 복원

  1. 복원할 기기에 PwNote를 설치합니다.
  2. 현재 마스터 비밀번호로 초기 설정을 완료합니다.
  3. 설정의 가져오기 / 복원으로 이동합니다.
  4. 파일 앱에서 암호화 백업 파일을 선택합니다.
  5. PwNote는 현재 마스터 비밀번호로 자동 복호화를 시도합니다. 백업 생성 당시 비밀번호와 다르면 복원에 실패합니다.
중요: 복원을 성공하려면 현재 마스터 비밀번호가 백업 생성 시 사용한 비밀번호와 같아야 합니다.

4. Pro 구매 및 구매 복원

4.1 Pro 구매

  1. PwNote에서 Pro / 업그레이드로 이동합니다.
  2. 기능을 확인하고 구매를 누릅니다.
  3. Apple ID로 인앱 구매를 확인합니다.
  4. 구매 성공 후 Pro 기능이 자동으로 열립니다.

4.2 새 기기에서 Pro 복원

  1. 새 기기(또는 재설치 기기)에서 PwNote를 엽니다.
  2. Pro / 업그레이드로 이동합니다.
  3. 구매 복원을 누릅니다.
  4. 원래 구매한 Apple ID 로그인 및 네트워크 연결을 확인합니다.
  5. 복원 완료 후 Pro 기능을 다시 사용할 수 있습니다.
참고: 복원 후에도 Pro가 잠금 해제되지 않으면 앱 재시작, Apple ID 확인, 네트워크 확인을 시도하세요.

5. 자주 묻는 질문

5.1 마스터 비밀번호를 잊었습니다. 재설정해 줄 수 있나요?

아니요. 보안상 당사는 마스터 비밀번호에 접근하거나 재설정할 수 없습니다. 복구 키로만 접근을 되찾을 수 있습니다.

5.2 마스터 비밀번호와 복구 키를 모두 잃어버렸습니다. 복구 가능할까요?

불가능합니다. 당사는 서버에 보관함을 저장하지 않으며 키를 알 수 없습니다.

5.3 기기 간 데이터 동기화를 지원하나요?

클라우드 동기화/온라인 계정을 사용하지 않습니다. 파일 앱의 암호화 백업 내보내기/가져오기를 이용하세요.

5.4 SSO와 OTP를 지원하나요?

SSO 로그인 및 TOTP 항목 저장을 지원합니다. 단, 외부 ID 제공자와 직접 연동하지는 않습니다.
