1. 概述

PwNote 是一款离线密码管理应用，设计目标是将你的密码与敏感备注保存在本地设备。 无需账号、没有云同步，你的保险库会通过主密码加密保护。

重要：PwNote 无法访问你的保险库、主密码或恢复密钥。 若同时遗失主密码和恢复密钥，数据将无法恢复。

2. 开始使用

2.1 首次启动

  • 首次打开 PwNote 时，请先设置主密码。
  • PwNote 会生成恢复密钥，并且仅显示一次。
  • 请将恢复密钥保存在安全位置（例如安全笔记、纸质备份或其他可信保险库）。
  • 完成后即可开始创建分类与条目（登录信息、备注等）。

2.2 免费版与 Pro 版

免费版限制：

  • 最多 5 个分类
  • 最多 30 条记录

Pro 版（App 内购）解锁：

  • 无上限条目（分类与保险库条目不受限）
  • Face ID 快速解锁
  • 密码生成器（创建高强度密码）
  • 重新生成恢复密钥（新密钥生成后，旧密钥失效）

3. 备份与恢复

PwNote 支持导出与导入加密备份文件。你可通过 iOS“文件”App 自行选择保存位置 （如 iCloud Drive 或其他位置）。

3.1 创建加密备份

  1. 打开 PwNote 并使用主密码解锁。
  2. 前往设置中的备份 / 导出区域（或可用的备份入口）。
  3. 选择导出加密备份。
  4. 在文件选择窗口中选择文件夹（如 iCloud Drive 或“我的 iPhone”）并保存文件。

3.2 从加密备份恢复

  1. 在目标设备上安装 PwNote。
  2. 打开 PwNote，完成初始化流程并设置当前主密码。
  3. 前往设置中的导入 / 恢复区域。
  4. 从“文件”App 选择加密备份文件。
  5. PwNote 会使用当前主密码自动尝试解密备份。 若当前主密码与创建备份时使用的密码不同，恢复会失败，数据无法载入。
重要：要成功恢复加密备份，你当前在 PwNote 中使用的主密码必须与导出备份时一致。 若你在创建备份后修改过主密码，恢复前需先改回原密码。

4. 购买 Pro 与恢复购买

4.1 购买 Pro

  1. 打开 PwNote，前往Pro / 升级区域（例如设置中）。
  2. 查看 Pro 功能后，点击购买按钮。
  3. 使用 Apple ID 确认 App 内购买。
  4. 购买成功后，Pro 功能应自动解锁。

4.2 在新设备恢复 Pro

若你重新安装 PwNote，或更换到使用同一 Apple ID 的新 iPhone，可恢复 Pro 购买：

  1. 在新设备或重装后的设备打开 PwNote。
  2. 前往Pro / 升级区域。
  3. 点击恢复购买。
  4. 确认已登录原购买所用 Apple ID，且网络连接正常。
  5. 恢复完成后，Pro 功能应可再次使用。
注意：若购买或恢复成功后仍未解锁 Pro，可先尝试：
  • 重启 PwNote
  • 确认 Apple ID 是否正确
  • 确认网络连接正常
若问题仍存在，请联系支持并提供购买日期与相关截图。

5. 常见问题

5.1 我忘记主密码了，可以帮我重置吗？

不可以。基于隐私与安全考虑，我们无法访问或重置你的主密码。 你只能通过恢复密钥重新获得访问权限。

5.2 我同时遗失主密码和恢复密钥，还能找回数据吗？

很遗憾，不能。因为我们不会在服务器保存你的保险库，也不知道你的密钥， 若同时遗失主密码和恢复密钥，数据将无法恢复。

5.3 PwNote 会在设备间自动同步吗？

PwNote 不提供云同步与在线账号。 如需在设备间迁移数据，请使用“文件”App 的加密备份导出/导入流程。

5.4 PwNote 支持 SSO 和 OTP 吗？

PwNote 可保存并管理使用 SSO 的登录信息，以及需要一次性密码（TOTP）验证的账号。 但 SSO 与 OTP 均以保险库条目形式保存；PwNote 不会直接连接外部身份提供方。
