1. 概述

PwNote 是離線密碼管理工具，設計目標是將你的密碼與敏感備註保存在本機裝置。 不需要帳號、沒有雲端同步，你的保管庫會以主密碼加密保護。

重要： PwNote 無法存取你的保管庫、主密碼或復原金鑰。 若同時遺失主密碼與復原金鑰，資料將無法復原。

2. 開始使用

2.1 第一次啟動

  • 首次開啟 PwNote 時請先設定主密碼。
  • PwNote 會產生復原金鑰，且只顯示一次。
  • 請將復原金鑰保存於安全位置（例如安全筆記、紙本備份或其他可信保管庫）。
  • 設定完成後即可開始建立分類與項目（登入資料、備註等）。

2.2 免費版與 Pro 版

免費版限制：

  • 最多 5 個分類
  • 最多 30 筆項目

Pro 版（App 內購）解鎖：

  • 無上限項目（分類與保管庫項目不受限）
  • 密碼產生器（建立高強度密碼）
  • 重新產生復原金鑰（新金鑰產生後，舊金鑰失效）

3. 備份與還原

PwNote 支援匯出與匯入加密備份檔。你可透過 iOS 檔案 App 自行選擇儲存位置 （例如 iCloud Drive 或其他位置）。

3.1 建立加密備份

  1. 開啟 PwNote 並使用主密碼解鎖。
  2. 前往設定中的備份 / 匯出區塊（或有提供備份的選單）。
  3. 選擇匯出加密備份。
  4. 在檔案選擇視窗中選擇資料夾（例如 iCloud Drive 或「我的 iPhone」）並儲存檔案。

3.2 從加密備份還原

  1. 在欲還原資料的裝置上安裝 PwNote。
  2. 開啟 PwNote，完成初始化流程並設定目前主密碼。
  3. 前往設定中的匯入 / 還原區塊。
  4. 從檔案 App 選擇加密備份檔。
  5. PwNote 會使用目前主密碼自動嘗試解密備份。 若目前主密碼與建立備份時使用的密碼不同，還原會失敗，資料無法載入。
重要：要成功還原加密備份，你在 PwNote 的目前主密碼必須與匯出備份時使用的主密碼一致。 若你在建立備份後更改過主密碼，還原前需先改回原本密碼。

4. 購買 Pro 與回復購買

4.1 購買 Pro

  1. 開啟 PwNote，前往Pro / 升級區塊（例如設定中）。
  2. 查看 Pro 功能後，點選購買按鈕。
  3. 使用 Apple ID 確認 App 內購買。
  4. 購買成功後，Pro 功能應會自動解鎖。

4.2 在新裝置回復 Pro

若你重新安裝 PwNote，或更換到使用相同 Apple ID 的新 iPhone，可回復 Pro 購買：

  1. 在新裝置或重裝後的裝置開啟 PwNote。
  2. 前往Pro / 升級區塊。
  3. 點選回復購買。
  4. 確認已登入原購買使用的同一個 Apple ID，且網路連線正常。
  5. 回復完成後，Pro 功能應可再次使用。
注意：若購買或回復成功後仍未解鎖 Pro，可先嘗試：
  • 重新啟動 PwNote
  • 確認 Apple ID 是否正確
  • 確認網路連線正常
若問題持續，請聯絡支援並提供購買日期與相關截圖。

5. 常見問題

5.1 我忘記主密碼了，可以幫我重設嗎？

不行。基於隱私與安全性，我們無法存取或重設你的主密碼。 你只能透過復原金鑰重新取得存取權。

5.2 我同時遺失主密碼與復原金鑰，還能救回資料嗎？

很遺憾，不行。因為我們不在伺服器保存你的保管庫，也不知道你的金鑰， 若同時遺失主密碼與復原金鑰，資料無法復原。

5.3 PwNote 會在裝置間自動同步嗎？

PwNote 不提供雲端同步與線上帳號。 若要在裝置間搬移資料，請使用檔案 App 的加密備份匯出/匯入流程。

5.4 PwNote 支援 SSO 與 OTP 嗎？

PwNote 可儲存並整理使用 SSO 的登入資料，以及需要一次性密碼（TOTP）驗證的帳號。 但 SSO 與 OTP 都是以保管庫項目形式保存；PwNote 不會直接連線外部身份提供者。
