1. Vue d'ensemble

PwNote est un gestionnaire de mots de passe hors ligne. Vos mots de passe et notes sensibles restent sur l'appareil. Aucun compte, aucune synchronisation cloud, et votre coffre est protégé par chiffrement via votre mot de passe maître.

Important : PwNote n'a pas accès à votre coffre, votre mot de passe maître ni votre clé de récupération. Si vous perdez les deux, les données ne peuvent pas être récupérées.

2. Démarrage

2.1 Premier lancement

  • Définissez votre mot de passe maître au premier lancement.
  • PwNote génère une clé de récupération affichée une seule fois.
  • Conservez cette clé en lieu sûr.
  • Après configuration, vous pouvez créer catégories et éléments.

2.2 Gratuit vs Pro

Limites de l'offre gratuite :

  • Jusqu'à 5 catégories
  • Jusqu'à 30 éléments

Pro (achat intégré) débloque :

  • Éléments illimités
  • Déverrouillage rapide Face ID
  • Générateur de mot de passe
  • Régénération de la clé de récupération (ancienne clé invalidée)

3. Sauvegarde et restauration

PwNote permet d'exporter/importer un fichier de sauvegarde chiffré. Vous choisissez l'emplacement via l'app Fichiers iOS.

3.1 Créer une sauvegarde chiffrée

  1. Ouvrez PwNote et déverrouillez avec le mot de passe maître.
  2. Allez dans Sauvegarde / Export des réglages.
  3. Choisissez Exporter une sauvegarde chiffrée.
  4. Choisissez un dossier dans Fichiers et enregistrez.

3.2 Restaurer depuis une sauvegarde chiffrée

  1. Installez PwNote sur l'appareil cible.
  2. Terminez l'initialisation avec votre mot de passe maître actuel.
  3. Allez dans Import / Restauration.
  4. Sélectionnez le fichier chiffré depuis Fichiers.
  5. PwNote tentera de déchiffrer avec le mot de passe maître actuel. S'il diffère de celui utilisé lors de l'export, la restauration échouera.
Important : le mot de passe maître actuel doit être identique à celui utilisé lors de l'export.

4. Achat Pro et restauration des achats

4.1 Acheter Pro

  1. Ouvrez la section Pro / Mise à niveau.
  2. Consultez les fonctions et appuyez sur Acheter.
  3. Confirmez l'achat via votre Apple ID.
  4. Une fois l'achat réussi, les fonctions Pro se débloquent automatiquement.

4.2 Restaurer Pro sur un nouvel appareil

  1. Ouvrez PwNote sur le nouvel appareil / après réinstallation.
  2. Allez dans Pro / Mise à niveau.
  3. Appuyez sur Restaurer les achats.
  4. Vérifiez que vous utilisez le même Apple ID et une connexion internet active.
  5. Après restauration, les fonctions Pro doivent revenir.

5. Questions fréquentes

5.1 J'ai oublié mon mot de passe maître. Pouvez-vous le réinitialiser ?

Non. Pour des raisons de sécurité, nous n'y avons pas accès. Vous devez utiliser votre clé de récupération.

5.2 J'ai perdu le mot de passe maître et la clé de récupération. Les données sont-elles récupérables ?

Non. Nous ne stockons pas votre coffre sur nos serveurs et ne connaissons pas vos clés.

5.3 PwNote synchronise-t-il entre appareils ?

Non. Utilisez l'export/import chiffré via l'app Fichiers.

5.4 PwNote prend-il en charge SSO et OTP ?

PwNote peut stocker les entrées SSO et OTP (TOTP), mais ne se connecte pas directement aux fournisseurs d'identité externes.
