1. 概要

PwNote はオフラインで動作するパスワードマネージャーです。パスワードや機密メモは端末内に保存されます。 アカウント登録やクラウド同期はなく、保管庫はマスターパスワードで暗号化保護されます。

重要: PwNote はあなたの保管庫、マスターパスワード、復旧キーにアクセスできません。 マスターパスワードと復旧キーの両方を失うと、データを復元できません。

2. はじめに

2.1 初回起動

  • 初回起動時に マスターパスワード を設定します。
  • PwNote は 復旧キー を生成し、1回のみ 表示します。
  • 復旧キーは安全な場所に保管してください。
  • 設定完了後、カテゴリや項目（ログイン情報、メモ等）を作成できます。

2.2 無料版と Pro

無料版の上限:

  • 5カテゴリまで
  • 30項目まで

Pro（アプリ内課金）で解放:

  • 無制限項目（カテゴリ・保管庫項目）
  • パスワードジェネレーター
  • 復旧キー再生成（旧キーは無効化）

3. バックアップと復元

PwNote は暗号化バックアップファイルのエクスポート／インポートに対応しています。 保存先は iOS のファイルアプリで選択します。

3.1 暗号化バックアップを作成

  1. PwNote を開き、マスターパスワードで解除します。
  2. 設定内の バックアップ / エクスポート に移動します。
  3. 暗号化バックアップをエクスポート を選択します。
  4. ファイルシートで保存先フォルダを選び、保存します。

3.2 暗号化バックアップから復元

  1. 復元先デバイスに PwNote をインストールします。
  2. 現在のマスターパスワードで初期設定を完了します。
  3. 設定の インポート / 復元 に移動します。
  4. ファイルアプリから暗号化バックアップファイルを選択します。
  5. PwNote は 現在のマスターパスワード で自動復号を試みます。バックアップ作成時のパスワードと異なる場合、復元は失敗します。
重要: 復元を成功させるには、現在のマスターパスワードがバックアップ作成時のマスターパスワードと同一である必要があります。

4. Pro 購入と購入の復元

4.1 Pro を購入する

  1. Pro / アップグレード セクションを開きます。
  2. Pro 機能を確認し、購入 をタップします。
  3. Apple ID でアプリ内課金を承認します。
  4. 購入完了後、Pro 機能は自動的に解放されます。

4.2 新しい端末で Pro を復元する

  1. 新しい端末（または再インストール後）で PwNote を開きます。
  2. Pro / アップグレード セクションに移動します。
  3. 購入を復元 をタップします。
  4. 元の購入時と同じ Apple ID でサインインし、ネット接続が有効であることを確認します。
  5. 復元完了後、Pro 機能が再び利用可能になります。

5. よくある質問

5.1 マスターパスワードを忘れました。リセットできますか？

できません。セキュリティ上、当社はマスターパスワードを確認・リセットできません。復旧キーを使ってのみアクセスを回復できます。

5.2 マスターパスワードと復旧キーの両方を失いました。復旧できますか？

申し訳ありませんができません。当社はサーバー上に保管庫を保存せず、キー情報も保持していません。

5.3 PwNote は端末間同期に対応していますか？

クラウド同期やオンラインアカウントは使用しません。暗号化バックアップのエクスポート／インポートをご利用ください。

5.4 PwNote は SSO と OTP に対応していますか？

SSO ログインや OTP（TOTP）を使うアカウント情報を保存できます。ただし外部IDプロバイダへ直接接続はしません。
