1. Panoramica

PwNote è un gestore password offline. Password e note sensibili restano in locale sul dispositivo. Nessun account, nessuna sincronizzazione cloud, e vault protetto da cifratura con Password principale.

Importante: PwNote non ha accesso al tuo vault, alla Password principale o alla Chiave di recupero. Se perdi entrambe, i dati non sono recuperabili.

2. Per iniziare

2.1 Primo avvio

  • Imposta la Password principale al primo avvio.
  • PwNote genera una Chiave di recupero e la mostra una sola volta.
  • Conserva la chiave in un luogo sicuro.
  • Dopo la configurazione, puoi creare categorie ed elementi.

2.2 Free vs Pro

Limiti piano gratuito:

  • Fino a 5 categorie
  • Fino a 30 elementi

Pro (acquisto in-app) sblocca:

  • Elementi illimitati
  • Generatore password
  • Rigenera Chiave di recupero (la precedente diventa non valida)

3. Backup e ripristino

PwNote permette esportazione/importazione di backup cifrati. Scegli tu la posizione tramite l'app File di iOS.

3.1 Creare un backup cifrato

  1. Apri PwNote e sblocca con la Password principale.
  2. Vai alla sezione Backup / Esporta nelle Impostazioni.
  3. Scegli Esporta backup cifrato.
  4. Nel foglio File, seleziona una cartella e salva.

3.2 Ripristinare da backup cifrato

  1. Installa PwNote sul dispositivo di destinazione.
  2. Completa la configurazione iniziale con la Password principale attuale.
  3. Vai alla sezione Importa / Ripristina.
  4. Seleziona il file di backup cifrato dall'app File.
  5. PwNote proverà a decifrare con la Password principale attuale. Se è diversa da quella usata all'esportazione, il ripristino fallirà.
Importante: per ripristinare con successo, la Password principale attuale deve essere la stessa usata durante l'esportazione del backup.

4. Acquisto Pro e ripristino acquisti

4.1 Acquistare Pro

  1. Apri la sezione Pro / Upgrade.
  2. Controlla le funzionalità Pro e tocca Acquista.
  3. Conferma l'acquisto in-app con il tuo Apple ID.
  4. Dopo l'acquisto, le funzioni Pro si sbloccheranno automaticamente.

4.2 Ripristinare Pro su un nuovo dispositivo

  1. Apri PwNote sul nuovo dispositivo o dopo reinstallazione.
  2. Vai alla sezione Pro / Upgrade.
  3. Tocca Ripristina acquisti.
  4. Verifica di usare lo stesso Apple ID dell'acquisto originale e di avere rete attiva.
  5. Al termine, le funzioni Pro torneranno disponibili.

5. Domande frequenti

5.1 Ho dimenticato la Password principale. Potete reimpostarla?

No. Per motivi di sicurezza non possiamo accedere o reimpostare la Password principale. Puoi recuperare l'accesso solo con la Chiave di recupero.

5.2 Ho perso Password principale e Chiave di recupero. Potete recuperare i miei dati?

Purtroppo no. Non archiviamo il tuo vault su server e non conosciamo le tue chiavi.

5.3 PwNote sincronizza i dati tra dispositivi?

No. Usa esportazione/importazione cifrata tramite app File.

5.4 PwNote supporta SSO e OTP?

PwNote può salvare credenziali SSO e OTP (TOTP), ma non si collega direttamente ai provider di identità esterni.
